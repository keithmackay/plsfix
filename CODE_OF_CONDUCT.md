# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

All participants are expected to uphold this standard. The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

To report a violation, contact the maintainer at the email listed in the GitHub profile for [keithmackay](https://github.com/keithmackay). All reports will be reviewed and investigated promptly and confidentially.
