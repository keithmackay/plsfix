# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest (main) | ✅ |

## Reporting a Vulnerability

plsfix is a markdown skill file with no executable code, no dependencies, and no network access. There is no attack surface in the traditional sense.

If you identify a way the skill could be used to generate harmful, misleading, or unsafe outputs — for example, a crafted input document that causes it to produce dangerous instructions — please report it responsibly rather than opening a public issue.

To report, contact the maintainer via the email listed in the GitHub profile for [keithmackay](https://github.com/keithmackay). Include:

- A description of the concern
- A minimal example input that reproduces it
- What harmful output was produced

You can expect an acknowledgment within 5 business days and a resolution or decision within 30 days.
